### 메모: 
- `@discardableResult`를 사용하게 되면 함수의 return값을 사용하지 않을 때 발생하는 오류를 지워주는 역할을 한다.
~~~ swift
@discardableResult func test(_ test:String) -> String { 
	return "Hello \(test)"
}

// 오류 안뜸
test("huni")
~~~
